STAT Summit

I developed a microsite for STAT Summit, an event for leaders in science and medicine.

I worked closely with STAT's Senior Product Designer to create a product that was true to her vision. I shipped code in small iterations to get fast feedback on animation, typography, and other topics.

I used the Intersection Observer API to update the navigtation's active state as the user scrolls through the site. This modern API allowed me to implement scroll effects without sacrificing performance.
